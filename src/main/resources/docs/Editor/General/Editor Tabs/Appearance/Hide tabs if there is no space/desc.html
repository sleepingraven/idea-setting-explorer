If this checkbox is selected, IntelliJ IDEA shows as many tabs as there fit into the current IntelliJ IDEA frame; the rest are hidden under the Hide tabs drop-down. Clear the checkbox if you don't want to hide any tabs. Alternatively, from the main menu, select Window | Editor Tabs | Show Hidden Tabs for IntelliJ IDEA to display the hidden tabs.

The Hide if there is no space checkbox becomes available when Show tabs in single row is selected.
