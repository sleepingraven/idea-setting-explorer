Use this list to select a bundled Maven version that is available (for Maven3, version 3.0.5 and later) or the result of resolved system variables such as MAVEN_HOME. You can also specify your own Maven version that is installed on your machine. You can click the Browse button and select the necessary directory in the dialog that opens.

(Maven2 is not bundled with IntelliJ IDEA. If you need to use it in your project, install it locally.)
