Use this page to manage plugins: browse the plugin repository, install, remove, enable, disable, and update plugins.

Use The Plugin Settings icon to configure additional plugin settings:

  • Manage Plugin Repositories: Open the Custom Plugin Repositories Dialog. For more information, see Custom plugin repositories.
  • HTTP Proxy Settings: Open the HTTP Proxy settings dialog.
  • Install Plugin from Disk: Select the local archive file (ZIP or JAR) from which to install a plugin.

Use the Custom Plugin Repositories dialog to manage the list of custom plugin repositories. The repositories are identified by their URLs.
