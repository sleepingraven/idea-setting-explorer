Use this list to define the IntelliJ IDEA behavior when a patch is created. The available options are:

  • Ask: if this option is selected, IntelliJ IDEA will display a dialog informing about successful patch creation, and a suggestion to locate the patch file in the native file manager.
  • Yes: if this option is selected, native file manager always opens to show the patch file.
  • No: if this option is selected, the native file manager will not open.
