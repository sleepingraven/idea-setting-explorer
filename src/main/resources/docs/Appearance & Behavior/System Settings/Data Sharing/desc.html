Use this page to share the statistics of your IntelliJ IDEA usage with JetBrains.

Select the Send usage statistics when using EAP versions checkbox to allow JetBrains to collect statistics on the features and actions that you use most frequently when working with IntelliJ IDEA in early access. Specifically, we collect information on:

  • Used features
  • Installed plugins
  • Programming languages
  • Frameworks
  • Number of files per project

Collecting and analysing feature usage statistics helps us greatly in IDE development. It gives us a picture of the most and least used actions so that we could focus on those aspects that you require most.

(We respect your privacy: all feature usage statistics that we receive from you is strictly anonymous.)
