Defines the behavior of the Save 'word' to dictionary intention action invoked on a mistyped word:

  • If the checkbox is selected, IntelliJ IDEA lets you select a single dictionary from the list. The selected dictionary is used for saving all mistyped words.
  • If the checkbox is cleared, IntelliJ IDEA lets you choose a dictionary in the code editor on the fly, individually for each a mistyped word.
