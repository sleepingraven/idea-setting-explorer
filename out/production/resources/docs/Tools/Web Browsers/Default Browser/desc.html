In this section, specify the default IntelliJ IDEA browser that will be used by default for rendering external resources and previewing files with Web contents. This browser will be referred to as Default in the context menu when you choose View | Open in Browser from the main menu or Open in Browser from the context menu of a file.

  • System default: Select this option to accept your operating system default Web browser as default for IntelliJ IDEA.
  • First listed: Select this option to have IntelliJ IDEA launch the first browser in the list.
  • Custom path: Select this option to specify another Web browser as default for IntelliJ IDEA.
