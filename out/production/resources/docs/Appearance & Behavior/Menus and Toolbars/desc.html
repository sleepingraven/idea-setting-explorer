Use this page to configure the IntelliJ IDEA menus and toolbars.

Menus and Items List shows the items for the menus and toolbars. The items are grouped according to the areas of their use.

To configure an item, expand the corresponding node and select the desired item. After that, the buttons in the right-hand part of the page become available.

(The image file should have .png or .svg extension.)

The Choose Actions to Add dialog opens when you select an item in the Menus and Items List and click the Add After button.

In the dialog, choose the desired action to be added to the menu or toolbar and optionally assign an icon to it.
